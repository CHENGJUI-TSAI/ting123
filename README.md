# 電機系資訊爬蟲 (單一檔案版本)

這個Python程式用來爬取國立虎尾科技大學電機工程系的最新動態和公告資訊，並提供網頁展示。所有程式碼整合在單一檔案中。

## 功能

- 爬取電機系網站的最新消息
- 顯示標題和鏈接
- 提供網頁界面顯示數據
- 內建聊天機器人功能

## 要求

- Python 3.x
- requests
- beautifulsoup4
- flask

## 安裝依賴

```bash
pip install -r requirements.txt
```

## 運行網頁應用

```bash
python single_app.py
```

然後在瀏覽器中打開 http://127.0.0.1:5000/ 來查看數據。

## 輸出

網頁應用會在本地伺服器上顯示可點擊的鏈接列表，並提供聊天功能。
